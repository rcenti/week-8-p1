Instructions

Use oscillators to create a musical instrument that generates sound when the user interacts with shapes on the canvas.

To create an oscillator, you will need to create an object of one of the p5.js sound library's oscillator classes: p5.SinOsc, p5.TriOsc, p5.SqrOsc, or p5.SawOsc. Here is an example: 

let osc;
function setup() {
    createCanvas(400, 300);
    osc = new p5.SinOsc();
}

To start an oscillator, call osc.start(). To stop an oscillator, call osc.stop().

Your instrument should play at least five different notes. You can create different notes using osc.freq() to set the frequency (pitch) of the oscillator. The higher the frequency, the higher the note.

In the video below, five different notes are generated randomly. The user clicks on a square to play the associated note. The shade of blue used to fill each square reflects the pitch of the note--the darker the fill, the lower the note. The x location of each square is used to set it's pan.

You will find the p5.js Oscillator documentation helpful for this task.

This is an open-ended creative task so there are no tests for this exercise.

Example output:

You will need to choose the frequency of each note. Picking frequencies between 50 and 1000 is recommended but you can go with whatever frequencies appeal to you.

If you would like to make your instrument sound like a real instrument e.g. a piano, try searching online for the frequencies of specific notes. For example, this resources lists the frequencies of notes on a keyboard: https://www.vibrationdata.com/tutorials2/piano.pdf.
