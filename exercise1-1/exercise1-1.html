Exercise 1.1

Your sketch:

Instructions

Download three sounds of your choosing from https://freesound.org or use the cat sounds provided in the assets folder. If you use your own sounds, put them in the assets folder.

Create a new sketch and load each sound in the preload() function. Use the examples from lecture as reference. You can also consult the p5.SoundFile documentation.

Play a sound when the user presses a key. Each sound should be associated with a different key. For example, in my solution, pressing 1 plays the meow sound, pressing 2 plays the purring sound, and pressing 3 plays the snoring sound. If a sound is already playing when the user presses a key, stop that sound before playing the new sound. If you're not sure how to do this, consult the p5.SoundFile documentation.

For an extra challenge, add a visualisation using p5.Amplitude or p5.FFT. The sample solution uses the p5.FFT waveform() method.

Check your work using check-exercise1-1.html.

Expected output:

You will need a global variable for each of the three sounds. Alternatively, you can use a single global array to store each sound.

Implement the preload() function and initialise the global variables by calling loadSound(). You will need to pass loadSount() the path of a sound file. If your files are store in the assets folder, the path will begin with "assets/".

Implement the keyPressed() event function and use it to play a sound when a key is pressed.
