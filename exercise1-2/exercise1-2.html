Exercise 1.2

Your sketch:

Instructions

One of the most common use of audio in digital media is to create sound effects. Your task is to enhance the bouncing ball sketch by adding a sound effect that plays every time the ball hits a wall. A "boing" sound file is already in the assets folder but feel free to use a different sound if you prefer.

You should only need to add a little code to complete this exercise! Remember that most browsers will not autoplay a sound so you will likely need to click the canvas before the sound effect begins to play.

Check your work using check-exercise1-2.html.

Expected output:

You will need a global variable for the sound, which should be initialised in the preload() function by calling loadSound().

The sound should play whenever the ball changes direction. The logic to determine when a direction change should take place has already been implemented. You just need to add a little code to play the sound at the same time.
